## Purpose

Project Helldiver is a specialized team of 7 AI agents that analyzes any arbitrary project, determines what to instrument, configures a Sauron hub instance to receive its telemetry, generates Grafana dashboards, and validates the integration end-to-end.

Helldiver inherits Voltron's infrastructure (Docker orchestration, Alexandria integration, reflection pipeline) and adds domain-specific observability agents.

**Sauron instance:** `https://sauron.7ports.ca` (Rajesh's personal hub)

## Environment Variables

See `.env.example`. Required:
- `SAURON_URL` — URL of the target Sauron instance (e.g., `https://sauron.7ports.ca`)
- `SAURON_PUSH_TOKEN` — Bearer token for this client (generated by Helldiver during onboarding)
- `GITHUB_TOKEN` — for reading target repos and pushing configs to project-sauron
- `TARGET_REPO` — GitHub URL or local path of the project to onboard
